# CPP Project
Simple dns laboratory for UTM

## Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/CalinCezar/CPP-UTM
```

2. Navigate to the project directory

```bash
cd CPP-UTM/lab_5
```

3. Create a virtual environemnt

```bash
python -m venv venv
```

4. Activate the virtual environment
for Linux/Mac
```bash
source venv/bin/activate

```
for Windows
```bash
venv\Scripts\activate

```
5. Install the dependencies

```bash
pip install -r requirements.txt
```

6. Install firefox and geckodriver

7. Put the paths of firefox and geckodriver in .env check .env-example
